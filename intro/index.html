An Intro to code

Code is a language like any other, it has grammar, pronouns, and verbs. The only thing that substantially differentiates the two type of language is that you will never learn a coding language completley. Almost Half of all of code is research, you always need to research what functions or methods you need to use to complete a specific task. This can act as an introduction to any of these languages.
